## Design

New `client/src/render/renderResolver.ts` — a thin stateful coordinator
`RenderResolver(stepMs)` exposing `resolve({characters, ownEntityId, predicted,
snapped, now}): RenderEntity[]` and `reset()`. The pure decision logic stays in
`SlideClock`/`interpolate`; the resolver is glue (functional-core/imperative-shell).

- **Own path** (used iff `ownEntityId !== undefined && entityId === ownEntityId &&
  predicted !== undefined`): lazily create the `SlideClock` at the current predicted
  tile; each frame `snapTo(tile, now)` if `snapped` else `setTarget(tile, now)`
  (same-tile re-affirm is a SlideClock no-op — the anti-stutter property); render
  `slideClock.positionAt(now)` (fractional). `action`/`facing` from `predicted`.
- **Interpolation path** (every other entity, incl. the own entity during the
  login/reconnect gap where `predicted`/`ownEntityId` is absent): `interpolate(c.prev,
  c.latest, now − interpDelayMs(stepMs))`. Returns integer cleanly when `prev` is
  undefined → no throw, no stale-deref. `action`/`facing` from `c.row`.
- **`reset()`** drops the own `SlideClock`; called from `main.ts onReconnect`
  alongside the existing `predictor = new Predictor(...)`.

`main.ts frame()`: sample `now` once; `const { snapped } = predictor.drain(now)`;
`renderer.render(resolver.resolve({ characters: store.characters(), ownEntityId:
store.ownEntityId(identity), predicted: predictor.predicted, snapped, now }))`;
delete the old `renderEntities()`. A sticky module-level `sawFractionalOwnMotion`
boolean is set true in `frame()` when the **own** resolved entity's x or y is
non-integer, exposed via `window.__game()`.

**Store `prev`:** already on `StoredCharacter` (`Snapshot` ≡ `interpolate`'s
`InterpSample`). Consumed by the interpolation path → **no `store.ts` edit** (smaller
diff; satisfies "consume or delete the prev snapshot").

## Proof-of-teeth (the gate)

`client/src/render/renderResolver.test.ts` — node-only, injected `now`, runs in
`just ci` via `client-test` (e2e is NOT in `just ci`). Must bite:
- **Own fractional mid-motion:** seed clock at tile 0, `setTarget` tile 1 at now=0
  (a real tile CHANGE so the slide starts), `positionAt(100)` (STEP_MS=200) ⇒
  x≈0.5, `Number.isInteger(x) === false`. A modeled raw renderer returning
  `predicted.pos.x` ⇒ integer 1 ⇒ fails the fractional assertion (the bite).
- **1-tile move stays fractional** (catches a snapTo-every-frame impl).
- **Remote fractional:** `prev`/`latest` with **distinct** `receivedAt`, renderTime
  strictly between ⇒ fractional; raw-latest-tile ⇒ integer (the bite).
- **Hold-not-extrapolate:** renderTime past `latest.receivedAt` ⇒ equals `latest`.
Keep `render/slideClock.test.ts` + `render/interpolation.test.ts` green (untouched).

## e2e (integration proof; ADR-0010)

`golden.spec.ts`: add `sawFractionalOwnMotion` to the `Snap`/`snap()` projection;
after the existing converged A-move, `waitForFunction(() =>
__game().sawFractionalOwnMotion)` in **A's own window**. Sticky + own-specific +
post-convergence ⇒ non-flaky (the ~60fps loop observes a fractional own frame over
the 200ms slide; a single own move suffices — own uses the slide clock, not the
two-snapshot interpolation path). **Fallback:** if remote CI shows flake, skip-mark
(never delete); the node gate remains load-bearing.

## Review adjudication (reviewer + red-team, plan stage)

**Adopted:** `predicted===undefined` guard (no deref/throw); own-path predicate;
biting test sequence (tile-change seed, 1-tile-fractional, distinct timestamps,
modeled raw bites); OWN-specific sticky e2e latch.

**Documented residual (out of touch-set):** reconcile's internal `drain` can consume
the `snapped` signal (predictor.ts:144), so a background→foreground with in-flight
moves may miss a snap (rare; a brief visual fast-slide vs instant-jump — NOT a desync;
predicted==auth holds). Robust fix = latch `snapped` across reconcile drains in
`predictor.ts`, which is **outside this slice's `touches:`** → deferred to the
predictor-owning slice. Honor `snapped` per spec here. A resolver-side distance-snap
heuristic is rejected (regresses legitimate chained held-key slides; YAGNI).

**Deferred per spec §4:** `speciesMap()/skillMap()` defensive copies + negative-`active`
guard ⇒ M8.6c. Reconnect row/sprite lifecycle + box re-signal ⇒ connection adapter
(out of touch-set), unchanged by this slice.
